React Recipes

React Recipes is my latest personal projects and It's my first approach to a MERN (MongoDB, Express, React, Node) application. The idea behind React Recipes was to develop a portal where registered users could share their recipes and rate/comment recipes of other users. I then decided to design an Universal App for experiment with some technologies like MongoDB, NodeJS and Server Side Rendering. It seemed natural to me then to use Next.js for React Recipes' develop. I really learnt a lot thanks to this project and if you want to learn more about, check the GitHub's repo



Tech Used

  • React
  • NextJs
  • ExpressJS
  • Mongoose
  • Redux
